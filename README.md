# Currency Converter

**Автор:** Arsentiev school

## Описание
Приложение с графическим интерфейсом для конвертации популярных валют с помощью внешнего API. Ведется история операций, которая сохраняется на вашем устройстве.

---

## Получение API-ключа

1. Перейдите на сайт [exchangerate-api.com](https://www.exchangerate-api.com/).
2. Нажмите "Get Free Key" или "Sign Up".
3. Зарегистрируйте бесплатную учетную запись, используя e-mail.
4. После подтверждения вы получите свой API-ключ на почту или увидите его на главной странице профиля.
5. Вставьте полученный ключ в переменную `API_KEY` в вашем файле `converter.py` (или там, где указан ключ). Например:

    ```python
    API_KEY = "ВСТАВЬТЕ_СВОЙ_API_КЛЮЧ_СЮДА"
    API_URL = f"https://v6.exchangerate-api.com/v6/{API_KEY}/latest/"
    ```

**Если вы используете бесплатные публичные API, как в примере кода, отдельный ключ не нужен.**

---

## Пример использования

1. Запустите файл:

    ```bash
    python converter.py
    ```

2. Введите сумму, выберите валюту "Из" и "В" (например, RUB ➔ USD).
3. Нажмите "Конвертировать".
4. Внизу отобразится результат и история прошлых операций.

## Возможные тесты

- Переведите 100 USD в EUR и убедитесь, что результат появляется в истории.
- Проверьте, что при вводе отрицательного числа появляется сообщение об ошибке.
- Смените пары валют и получите новые результаты.
- Перезапустите приложение — история ваших конвертаций сохранится.
